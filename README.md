Hospital Patient Data Analysis using SQL

A SQL practice project built using a **Hospital Patient Management** dataset. This project demonstrates the use of SQL aggregate functions, `GROUP BY`, `HAVING`, filtering, sorting, and subqueries to analyze patient records and generate business insights.

## Project Overview

This project contains a dataset of **30 hospital patient records** with information such as:
- Patient Details
- Department
- Diagnosis
- Admission & Discharge Dates
- Bill Amount
- Insurance Status
- Doctor
- Patient Rating

The project focuses on solving real-world SQL analytical questions using aggregate functions and grouping techniques.

## 📈 Business Questions Solved

This project answers 40 analytical questions, including:

- Count patients by department
- Calculate total and average hospital bills
- Identify departments with the highest revenue
- Analyze patient distribution by city
- Compare insurance vs non-insurance billing
- Find top-performing doctors based on revenue
- Calculate average patient ratings
- Identify departments with high patient volume
- Compare city-wise hospital expenses
- Perform aggregate filtering using HAVING
- Compare grouped averages with overall averages using subqueries
